# 🤖 Guía: Gentle-AI + Claude Code en Windows 11 Nativo

> Guía gratuita y open-source para potenciar tu workflow de desarrollo con IA.

## 📖 ¿Qué es esta guía?

Esta guía te enseña a instalar y configurar **Claude Code** (el agente de coding de Anthropic) potenciado con el **ecosistema Gentle-AI**, todo de forma **100% nativa en Windows 11**.

**Sin WSL2. Sin Node.js. Sin dependencias complejas.** Solo Git for Windows y dos comandos.

## 🎯 ¿Qué vas a conseguir?

| Sin Gentle-AI | Con Gentle-AI |
|---------------|---------------|
| ❌ Sin memoria entre sesiones | ✅ **Engram** — memoria persistente en SQLite |
| ❌ Desarrollo ad-hoc | ✅ **SDD** — workflow estructurado de 9 fases |
| ❌ Documentación con fecha de corte | ✅ **Context7** — docs de librerías en tiempo real |
| ❌ Code review solo si lo pedís | ✅ **GGA** — review automático en cada commit |
| ❌ Explicás todo de nuevo | ✅ Claude ya conoce tu proyecto |

- Troubleshooting específico de Windows
